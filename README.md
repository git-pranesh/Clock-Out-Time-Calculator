# Clock Out Time Calculator

A free tool to calculate what time you can leave work, given when you started and how many hours you need to work.

Live: https://www.calculatorhq.com

## What it does

Enter your clock-in time and your required shift length, and the tool tells you exactly when you can clock out. Handles breaks, overnight shifts, and different work schedule formats. Includes SEO-optimized pages for common search queries like "what time do I get off work" and "8 hour shift calculator".

## Stack

- React
- TypeScript
- Vercel (deployment)
- Built on Replit

## Project structure

```
artifacts/        Static data and SEO page configs
lib/              Time calculation logic
scripts/          Build and utility scripts
attached_assets/  Static assets
```

## Running locally

```bash
pnpm install
pnpm run dev
```
